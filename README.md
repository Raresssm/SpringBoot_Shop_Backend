# 🛒 SpringBoot Shop Backend

A RESTful e-commerce backend built with **Spring Boot**, **Spring Data JPA**, and **H2 Database**.  
This project provides product management functionality including image upload and retrieval.

---

## 🚀 Features

✅ Create products with image upload  
✅ Retrieve all products  
✅ Retrieve product by ID  
✅ Retrieve product image separately  
✅ H2 in-memory database  
✅ JPA/Hibernate persistence  
✅ Multipart file handling  
✅ Clean layered architecture (Controller → Service → Repository)

---

## 🏗️ Tech Stack

| Layer | Technology |
|--------|------------|
| Framework | Spring Boot |
| ORM | Spring Data JPA |
| Database | H2 (In-Memory) |
| Build Tool | Maven |
| File Upload | MultipartFile |
| Boilerplate Reduction | Lombok |
